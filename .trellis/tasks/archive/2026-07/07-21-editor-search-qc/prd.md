# PRD：编辑页字幕查找替换、筛选与质检

## 背景

编辑页已有丰富格式功能，但缺少面向长字幕文件的导航与质量控制：

- `SubtitleList.tsx` 只有列表、选择与右键编辑操作，没有文本搜索或筛选。
- 未知样式只在列表中变色，没有统一的问题面板。
- 压制导出条件仅有「存在视频、存在字幕、输出路径有效」，无任何字幕质量提示。

## 需求

### R1 查找与替换

- R1.1 搜索范围：当前物理 Dialogue 行的 `primaryText`（编辑器唯一的物理行模型，不做双语分支）。
- R1.2 支持筛选条件：按样式（下拉选择）、仅空文本、时间范围（起止，`H:MM:SS.cc`，复用 `src/utils/timeInput.ts`）。
- R1.3 操作：查找上一处/下一处（循环）、替换当前、全部替换。
- R1.4 「全部替换」必须是一次项目历史操作（单条 undo 记录，可整体撤销/重做）。
- R1.5 替换只改 `primaryText`，不得触碰时间、样式、name/margins/effect 等其他字段。
- R1.6 查找命中后选中并滚动到对应行，同时将播放头移动到该字幕的开始时间；质检问题定位遵循同一行为。
- R1.7 提供 Ctrl+F 快捷键打开查找面板（不占用已被占用的快捷键）。

### R2 筛选

- R2.1 筛选条件与 R1.2 共享（样式 / 空文本 / 时间范围 / 搜索词），作用于查找范围；是否同时过滤列表显示为可选增强，不做强制。

### R3 字幕质检（纯前端规则）

全部默认作为**警告**展示，不阻止保存或压制：

- R3.1 空字幕（`primaryText` 去除 ASS 标签与空白后为空）。
- R3.2 结束时间不大于开始时间。
- R3.3 超出视频时长（视频时长未知时跳过该规则）。
- R3.4 与其他行时间重叠（一律警告）。**不**将相邻行「间隔过近」列为问题——短间隔在多数对轴场景下合理。
- R3.5 每秒字符数（CPS）过高（阈值常量，如 > 20）。
- R3.6 单行过长（如单行 > 42 字符）或换行过多（如 > 2 行）。
- R3.7 引用不存在的样式（与列表现有变色逻辑同源：`assStyles` 非空且 cue.style 不在其中）。

### R4 问题面板

- R4.1 统一的问题列表面板：展示 R3 全部规则的命中项（规则名 + 行号 + 简述）。
- R4.2 点击问题项选中并滚动到对应字幕行。
- R4.3 仅警告级别；不改变保存、压制流程的任何行为。
- R4.4 查找/质检面板在右侧窄列内须可正常阅读：整体纵向组织；「查找/替换为」「时间起/时间止」及成组操作按钮可各用一行两列，其余控件保持全宽或独占一行。
- R4.5 展开后的头栏与内容须作为同一悬浮卡片覆盖字幕列表，不压缩列表高度；整卡最大高度取视口高度 60% 与 480px 的较小值，超出内容在卡片内部滚动。
- R4.6 点击头栏或 Ctrl+F 均打开并聚焦查找输入框；Esc 关闭卡片且不把焦点回收到头栏，避免关闭后出现醒目的焦点边框。

### R5 列表重叠高亮（Aegisub 风格）

- R5.1 当存在当前活动选中行（`selectedCueId`）时，凡与其时间区间**严格重叠**（`a.start < b.end && b.start < a.end`）的其他物理行，在字幕列表中整行以醒目的危险/重叠样式高亮（类似 Aegisub 将重叠行标红）。
- R5.2 选中行本身保持原有选中样式；未知样式列着色与重叠高亮可并存（行级重叠优先于普通 hover，选中优先于重叠）。
- R5.3 多选时仅以活动行（`selectedCueId`）为重叠参照，不为每个 multi-select id 分别高亮（保持实现简单、与 Aegisub 单焦点一致）。
- R5.4 纯展示，不写入历史、不改保存/压制。

## 非目标

- 正则搜索、大小写切换之外的高级搜索语法（可不含，保持最小实现：普通子串匹配，默认忽略大小写）。
- 后端/Rust 侧质检；压制页不增加阻断逻辑。
- 自动修复功能。
- 「相邻间隔过近」质检规则。

## 验收标准

1. 点击头栏或按 Ctrl+F 均可打开查找面板并聚焦查找框；输入关键词可循环定位上一处/下一处，行被选中、滚动可见且播放头移动到该行开始时间；Esc 可关闭面板。
2. 替换当前仅替换当前选中命中行；全部替换后单次 Ctrl+Z 可整体还原。
3. 样式/空文本/时间范围筛选生效于查找与替换范围。
4. 问题面板列出全部 R3 规则命中项（**不含**间隔过近），点击可定位；空文档/无问题时显示空态。
5. 质检全部为警告：保存与压制流程无任何新增阻断。
6. 查找/质检面板在默认右栏宽度下字段与文案可读；展开为不挤压字幕列表的悬浮卡片，整卡不超过 `min(60vh, 480px)`，内容超出时内部滚动。
7. 选中某行时，与其时间重叠的其他行在列表中整行高亮为重叠态；取消选中后高亮消失。
8. `pnpm test` 与 `pnpm build` 通过。
