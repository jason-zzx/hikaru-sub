# 更新日志

## [0.2.0] - 2026-07-17

### 新增

- 字幕编辑器改为左侧视频与时间轴、右上字幕列表、右下样式与文本编辑器的三分区工作区。
- 支持拖动左右分隔条和右侧上下分隔条调整面板比例；布局会全局保存，双击任一分隔条可恢复默认比例。
- 字幕列表改为带吸顶表头的单行列布局，固定显示序号、开始、结束、样式和文本，并按文档内容显示 Layer、Name、MarginL、MarginR、MarginV 与 Effect。
- 支持对多选字幕批量应用样式、字体、字号、粗体、斜体、下划线、删除线、颜色、描边、阴影和对齐方式，一次操作可整体撤销或重做。
- 字幕整行复制、剪切和粘贴改用系统剪贴板与标准 ASS `Dialogue:` 行，可与 Aegisub 等外部工具互通；粘贴普通文本时会按行创建连续字幕。

### 改进

- 编辑器按物理 ASS `Dialogue:` 行一一加载、编辑、预览和保存；双语合并方式只在翻译生成字幕时生效，不再改变编辑阶段的行结构。
- 统一所有持久字幕修改的撤销与重做历史，覆盖正文、时间、样式、行内格式、增删行、批量操作及剪贴板操作，并保留多选、活动行和文本光标上下文。
- 字幕正文连续输入、Backspace 与 Delete 按 Aegisub 式分组；IME 输入保持实时预览，仅在提交后写入历史。
- 时间输入统一为 `H:MM:SS.cc`，按当前可见数字位置处理光标与编辑。
- 翻译页始终从转录 ASS 载入翻译源；仅进入页面不会覆盖或删除已有翻译字幕。

### 修复

- 保存和剪贴 ASS 字幕时完整保留 Layer、Name、MarginL、MarginR、MarginV 与 Effect，避免编辑往返后丢失事件字段。
- 完善保存检查点与脏状态判断，撤销或重做到最近一次成功保存的版本时可正确恢复“已保存”状态。

### 开发

- 引入 Trellis 项目规范、任务工作流和多代理开发配置，并补充字幕编辑、ASS 事件、剪贴板、历史记录、列表列布局及面板缩放测试。
- CI 与发布工作流的 Node.js 版本升级至 22。
- 更新 `actions/checkout`、`actions/setup-node` 与 `pnpm/action-setup`，替换已弃用的 GitHub Actions，并临时屏蔽 Rust 缓存动作的 `punycode` 弃用警告。

## [0.1.0] - 2026-07-15

### 新增

- 支持下载或导入视频、片段切取、本地日语转录、批量翻译、字幕校对和成片输出。
- 支持 faster-whisper、kotoba-faster-whisper、Parakeet 和 Qwen3-ASR，并按需准备运行时依赖与模型。
- 支持 ASS 双语字幕、样式编辑、libass 预览、音频波形和多泳道时间轴。
- Windows 提供 NSIS 安装包和 portable zip 两种发行形式。

### 重要说明

- 当前仅发布 Windows 构建，macOS 与 Linux 暂未开放下载。
- Windows 构建暂未代码签名，首次启动时可能显示 Microsoft SmartScreen 提示。
- 发布包不捆绑 FFmpeg、Python 3.11、ASR Python 依赖或模型权重；相关依赖在用户确认后按需准备。
